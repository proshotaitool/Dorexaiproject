# **App Name**: ProShot AI Toolkit

## Core Features:

- Image Compression: Compress images to reduce file size for faster loading and sharing.
- AI Background Remover: Automatically remove backgrounds from images using AI to isolate subjects.
- Text Summarization: Summarize large text into shorter, more concise versions using AI tool.
- PDF Conversion: Convert files to and from PDF format, including JPG, Word, and HTML.
- Meme Generation: Create memes by adding custom text to images.
- User Authentication: Secure user login and registration with role-based access control for standard users and admin accounts.
- Admin Panel: Admin panel to let super admin user to make real-time management of content, tools, and global settings. ## Style Guidelines:

- Primary color: Strong blue (#3F51B5) to convey trust and stability.
- Background color: Light gray (#F0F2F5) for a clean, modern look.
- Accent color: Vibrant purple (#7E57C2) to draw attention to important CTAs and interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif font for a modern, neutral feel. It will be used both for headlines and body text.
- Card-based layout with rounded corners and subtle shadows to create a visually appealing and organized interface.
- Consistent and clear icons for each tool category to aid in navigation and recognition.
- Smooth transitions and subtle animations to enhance user experience and provide visual feedback.
